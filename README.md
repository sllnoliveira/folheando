# 📖 Folheando - Gerenciador de Leituras

O **Folheando** é uma aplicação Full Stack de diário de leitura pessoal. Ele permite aos usuários gerenciar seus livros, atualizar o status de leitura, dar notas de 1 a 5 e escrever resenhas com a data de registro salva automaticamente.

---

## 🎨 Design e Interface
A interface do usuário foi planejada com um design limpo, moderno e personalizado com tons suaves de rosa, deixando de lado o visual padrão para ter uma identidade própria e delicada.

---

## 🛠️ Tecnologias Utilizadas

### Front-end:
- **HTML5** & **CSS3** (Design responsivo e personalizado com tons de rosa)
- **JavaScript (ES6+)** (Comunicação assíncrona com o servidor usando `fetch`)

### Back-end & Banco de Dados:
- **Python** (Linguagem principal do servidor)
- **Flask** & **Flask-CORS** (Criação da API que conecta o site ao banco)
- **MySQL** (Banco de dados relacional para salvar usuários, livros e leituras)

---

## 📁 Estrutura do Projeto
- `/frontend`: Contém a interface do sistema web (HTML, CSS e JavaScript).
- `/versao_terminal`: Pasta contendo os códigos iniciais do sistema rodando via terminal (histórico do projeto).
- `app.py`: Arquivo principal do servidor em Flask.
- `requirements.txt`: Lista de bibliotecas usadas no Python.
